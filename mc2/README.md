## mc2 sub-project stack
* JAVA 8
* TestNG
* Rest-assured
* Allure rest assured
* Allure
* Gradle

## Tests Location
Tests are stored in `mc2/src/main/tests/`

### Report
All test results will be collected in Allure report.
You can find Allure report by path
```
web_automation/build/allure-report/index.html
```
